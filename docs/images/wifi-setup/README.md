# WiFi 設定手冊截圖

[employee-wifi-setup-guide.md](../../employee-wifi-setup-guide.md) 引用的截圖。

## 待補截圖清單

依 OS 分類，請拍真實裝置的截圖，遮住密碼後存到對應檔名：

### Android（從 POCO F5 Pro 或 Pixel 拍最好）

| 檔名 | 內容 |
|------|------|
| `android-step1-ssid-list.png` | WiFi 設定頁，可以看到 `MDS-01` SSID |
| `android-step2-eap-form.png` | 點 SSID 後跳出的設定對話框，所有欄位可見 |
| `android-step3-connected.png` | 連線成功後 WiFi 詳細資訊（顯示 IP）|

### iOS

| 檔名 | 內容 |
|------|------|
| `ios-step1-credentials.png` | 連線時跳出的帳號/密碼欄位 |
| `ios-step2-trust-cert.png` | 伺服器憑證信任視窗 |

### Windows

| 檔名 | 內容 |
|------|------|
| `windows-add-network.png` | 「新增網路」對話框 — Win 11 把所有欄位（網路名稱 / 安全性類型 / EAP 方法 / 驗證方法 / 識別碼）塞在同一個 form |

### macOS

| 檔名 | 內容 |
|------|------|
| `macos-step1-credentials.png` | Wi-Fi 加入網路時的帳密輸入視窗 |

## 截圖規範

- **遮住密碼**：用黑色色塊遮住密碼欄位
- **遮住 hostname**：如果 status bar 有顯示主機名，遮掉
- **PNG 格式**：解析度建議 1080×2400 (Android) / 1170×2532 (iPhone) 原生大小
- **檔案大小**：壓縮到 200KB 以下（用 [TinyPNG](https://tinypng.com/) 或 `pngquant`）
